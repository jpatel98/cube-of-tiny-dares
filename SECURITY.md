# Security Policy

Cube of Tiny Dares is a small hackathon project, but security reports are still welcome.

## Supported versions

Only the latest `main` branch is supported.

## Reporting a vulnerability

Please do **not** open a public issue for sensitive security problems.

Email: hello@axiontechnologies.ca

Include:

- affected file/endpoint
- reproduction steps
- impact
- suggested fix, if known

## Scope notes

The app does not require secrets for the MVP. If you add integrations later, never commit API keys, Wi-Fi credentials, or private endpoints.
